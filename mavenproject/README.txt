	--The Tvestor DataBase management application--

Before running this application, you must set up the database:
1. Open a terminal in the ../database folder
2. In the terminal execute the following command "java -cp hsqldb.jar org.hsqldb.Server --database test"

Now you can start the application. Do to that, run Tvestor.java as the main class of the project.
To execute tests, run AppTest.java as the main class of the project.

The application itself is comprised of 6 panels each of which corresponds to the 6 actions that the user can make as described in the assignment.
